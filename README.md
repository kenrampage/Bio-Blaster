# SSGJT2022
Shared repo for the ScoreSpace Jam #17

Bio Blaster is a Space FPS where you play as a Bio Ship pilot tasked with protecting a giant space beast from hostile contagions.

Featuring original music, sound effects, artwork, modeling and level design. All assets are made with love from scratch. Built in Unity Engine.
